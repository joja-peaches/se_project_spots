# Project 3: Spots

A simple image sharing platform.

## Description

Spots is a platform meant for web or mobile where users can share images, create and update a personal profile, and interact with other user's content via the like button.

## Tech Stack
* HTML
* CSS 
* Figma  
* Responsive Design
